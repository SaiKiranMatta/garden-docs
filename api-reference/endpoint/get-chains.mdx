---
title: 'Supported Chains'
openapi: 'GET /chains'
version: 1.0
--- 

Retrieve a list of all blockchain networks supported by the protocol. This endpoint returns detailed information about each supported chain, including network identifiers, and configuration details.

Use this endpoint to:
- Discover which blockchain networks are available for swapping
- Get chain-specific configuration and identifiers
- Validate chain compatibility before creating orders
- Access network metadata for integration purposes

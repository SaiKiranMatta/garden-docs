---
title: 'Supported Assets'
openapi: 'GET /assets'
version: 1.0
--- 

Retrieve a comprehensive list of all assets supported across different blockchain networks. This endpoint returns asset metadata including symbols, addresses, decimals, and availability by chain.

Use this endpoint to:
- Discover which tokens are available for swapping
- Get asset contract addresses and metadata
- Check asset availability on specific chains
- Validate asset parameters before creating swap orders

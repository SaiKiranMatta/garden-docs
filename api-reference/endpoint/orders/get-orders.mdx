---
title: 'Orders'
openapi: 'GET /orders'
version: 1.0
--- 

Retrieve a list of orders with optional filtering and pagination. This endpoint returns order details including status, amounts, assets, and transaction information.

Use this endpoint to:
- View all orders associated with your account
- Filter orders by status, asset, or date range
- Track order history and performance
- Monitor active and completed swaps

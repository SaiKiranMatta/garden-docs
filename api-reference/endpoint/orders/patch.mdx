---
title: 'Execute HTLC Action'
openapi: 'PATCH /orders/{order}'
version: 1.0
--- 

Execute Hash Time-Locked Contract (HTLC) actions on an existing swap order. This endpoint allows you to perform atomic swap operations such as initiate, redeem, refund and instant refund.

Use this endpoint to:
- Initiate an atomic swap, by sending the typed signature
- Redeem an atomic swap, by sending the secret 
- Refund an atomic swap after the timelock expires
- Execute an instant refund by sending the instant refund signature

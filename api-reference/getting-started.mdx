title: "Quickstart"
description: "Learn how to use Garden's API to get quotes, submit orders, and track their status. This guide provides a quick overview of the core API endpoints."

<Note>
    This guide uses a test API Key, please reach out to us over [discord](https://discord.gg/dZwSjh9922) for a production key.
</Note>

<Tabs>
  <Tab title="from bitcoin">
<Steps>
  <Step title="Get Quote">
    Let's get a quote to trade BTC on **Bitcoin Testnet4** to 0.001 WBTC on **Ethereum Sepolia**.
    ```bash wrap
    curl -X 'GET' 'https://testnet.api.garden.finance/quote?from=bitcoin:btc&to=ethereum:wbtc&amount=100000&exact_out=true' \
    -H 'accept: application/json'
    ```
    Try it out [here](/api-reference/endpoint/quote/get-quote?playground=open)
  </Step>
  <Step title="Submit Order">
    ```bash
    curl --location 'https://api.garden.finance/v1/orders' \
        --header 'api-key: 963ebcf89e26bd7d41c9adef039dd167584c371fb24f11f8adf1df0c7de3340f' \
        --header 'Content-Type: application/json' \
        --data '{
            "source": {
                "chain": "ethereum:wbtc",
                "owner": "0x004Cc75ACF4132Fc08cB6a252E767804F303F729",
                "amount": "50000"
            },
            "destination": {
                "chain": "bitcoin:btc",
                "owner": "bc1p4pr78swsn60y4ushe05v28mqpqppxxkfkxu2wun5jw6duc8unj3s9tp6jz",
                "amount": "49850"
            }
    }'
    ```
    Try it out [here](/api-reference/endpoint/orderbook/create-order?playground=open)
